division_by_zero/0

division_by_zero() -> any()

Division by 0.

When this occurs, operation signals an error result, containing infinity as fallback value. Infinity value is a bigdec element with value as atom inf.

division_impossible/0

division_impossible() -> any()

Cannot perform the division adequately.

When this occurs, operation signals invalid operation if the integer result of a divide-integer or remainder operation had too many digits that would be longer than precision is capable of calculating.

division_undefined/0

division_undefined() -> any()

Undefined result of division.

This occurs and signals invalid operation if division by zero was attempted, and dividend is also zero.

invalid_option/0

invalid_option() -> any()

Invalid options.

This occurs and signals invalid operation if an invalid option for context handling was detected during an operation. This might happen for example if unknown rounding is reached.
